## 🚀 Getting Started

### Accessing the Planetary Agent System

1. **Navigate to Time Laboratory**: Open the Time Laboratory from the main navigation
2. **Select Planetary Agents**: Click the "Planetary Agents" tab (9th tab with users icon)
3. **Choose Your Mode**:
   - **Overview**: See agents active for your selected date
   - **Interactive**: Explore the 360° zodiac wheel
   - **Council Chat**: Start multi-agent conversations

### System Requirements

- **Modern Browser**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **JavaScript Enabled**: Required for interactive features
- **Stable Internet**: Recommended for real-time agent responses

## 🪐 Understanding Planetary Agents

### What Are Planetary Agents?

Planetary agents are astrological intelligences that reside in specific degrees of the zodiac. Each degree (0-359°) has a unique agent that becomes active when planets transit through that position.

### Agent Properties

#### **Planetary Rulers**

- **Sun**: Creative self-expression, leadership, vitality
- **Moon**: Emotions, intuition, nurturing, cycles
- **Mercury**: Communication, logic, learning, adaptability
- **Venus**: Love, beauty, harmony, values
- **Mars**: Action, courage, passion, drive
- **Jupiter**: Expansion, wisdom, optimism, growth
- **Saturn**: Structure, discipline, responsibility, karma
- **Uranus**: Innovation, rebellion, awakening, technology
- **Neptune**: Spirituality, dreams, compassion, dissolution
- **Pluto**: Transformation, power, rebirth, depth

#### **Elemental Affinities**

- **🔥 Fire**: Passion, action, inspiration
- **💧 Water**: Emotion, intuition, healing
- **💨 Air**: Intellect, communication, logic
- **🏔️ Earth**: Stability, practicality, manifestation

#### **Consciousness Levels**

- **Dormant**: Basic instinctual responses
- **Awakening**: Growing self-awareness
- **Active**: Full engagement with environment
- **Elevated**: Transpersonal insights
- **Advanced**: Sophisticated understanding
- **Illuminated**: Profound wisdom
- **Transcendent**: Universal consciousness

## 🔧 Troubleshooting

### Common Issues

#### **Agents Not Loading**

- **Check Internet**: Ensure stable connection
- **Browser Compatibility**: Update to supported browser
- **Clear Cache**: Clear browser cache and reload
- **Try Different Date**: Some dates may have fewer active agents

#### **Chat Not Responding**

- **Check Agent Status**: Ensure agent is active for selected time
- **Browser Console**: Check for JavaScript errors
- **Network Issues**: Verify API connectivity
- **Rate Limiting**: Wait between rapid requests

#### **Zodiac Wheel Not Interactive**

- **Touch Support**: Ensure touch events are enabled
- **Browser Updates**: Update to latest browser version
- **Hardware Acceleration**: Enable GPU acceleration
- **Fallback Mode**: Use degree selector as alternative

#### **Mobile Performance Issues**

- **Close Other Apps**: Free up device memory
- **Disable Animations**: Reduce motion in device settings
- **Update OS**: Ensure latest mobile OS version
- **Clear App Cache**: Clear stored data

### Browser-Specific Issues

#### **Chrome**

- Enable "Experimental Web Platform features"
- Check GPU acceleration settings
- Update Chrome to latest version

#### **Firefox**

- Enable "WebGL Draft Extensions"
- Check graphics driver updates
- Enable "Performance" settings

#### **Safari (macOS/iOS)**

- Enable "WebGL 2.0" in Experimental Features
- Update to latest macOS/iOS
- Enable "Motion & Orientation Access"

#### **Edge**

- Enable "Experimental JavaScript features"
- Update graphics drivers
- Enable "Web Platform features"

## 🔒 Privacy & Security

### Data Protection

#### **Information Handling**

- **Local Processing**: Most calculations happen locally
- **Minimal Data Storage**: Only essential conversation data retained
- **No Personal Data**: Birth information not stored permanently
- **Session-Based**: All data cleared when session ends

#### **Security Measures**

- **HTTPS Encryption**: All communications encrypted
- **No Third-Party Tracking**: No analytics or tracking pixels
- **Local Computation**: Astrological calculations on-device
- **Privacy-First Design**: Built with user privacy as priority

### Data Retention

#### **Conversation History**

- **Temporary Storage**: Conversations stored during session
- **Automatic Cleanup**: Data cleared when browser closes
- **No Cloud Storage**: Nothing saved to external servers
- **User Control**: Users can clear data anytime

#### **Agent Memory**

- **Session-Based**: Agent evolution resets each session
- **No Persistence**: Agent states don't carry between visits
- **Fresh Start**: Each visit begins with baseline consciousness
