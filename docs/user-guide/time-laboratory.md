## Core Concepts

### Natal Charts

Your **natal chart** represents the celestial positions at your birth moment. It serves as the foundation for all transit analysis.

**Key Elements:**

- **Planets**: Sun, Moon, Mercury, Venus, Mars, Jupiter, Saturn, Uranus, Neptune, Pluto
- **Houses**: 12 astrological houses representing life areas
- **Aspects**: Angular relationships between planets
- **Signs**: Zodiac signs and their elemental qualities

### Transits

**Transits** occur when current planetary positions interact with your natal chart positions.

**Types of Transits:**

- **Conjunctions**: Planets align (0°)
- **Sextiles**: Harmonious 60° aspects
- **Squares**: Challenging 90° aspects
- **Trines**: Flowing 120° aspects
- **Oppositions**: Balancing 180° aspects

### Planetary Agents

**Planetary Agents** are AI-powered consciousness entities that provide personalized insights based on astrological transits.

**Agent Types:**

- **Historical**: Wisdom from figures like Leonardo da Vinci, Carl Jung
- **Planetary**: Direct planetary intelligence and guidance
- **Consciousness**: Evolutionary and spiritual coaching
- **Specialized**: Domain-specific expertise (tarot, runes, etc.)

### Significance Scoring

Each transit receives a **significance score** (0-1) indicating its potential impact:

| Score Range | Impact Level | Description                                 |
| ----------- | ------------ | ------------------------------------------- |
| 0.8 - 1.0   | Critical     | Life-changing events, major transformations |
| 0.6 - 0.8   | Major        | Important developments, significant changes |
| 0.4 - 0.6   | Moderate     | Notable influences, learning opportunities  |
| 0.2 - 0.4   | Minor        | Subtle influences, background themes        |
| 0.0 - 0.2   | Background   | Minimal impact, contextual information      |

## Transit Analysis

### Reading Transit Reports

Each transit includes detailed analysis:

#### Basic Information

- **Planets Involved**: Transiting and natal planets
- **Aspect Type**: Conjunction, sextile, square, trine, opposition
- **Exact Date/Time**: When the transit is most powerful
- **Orb**: How close the aspect is (smaller = stronger)

#### Significance Analysis

- **Overall Score**: 0-1 impact rating
- **Duration**: How long the transit's influence lasts
- **Peak Period**: When the transit is strongest
- **Themes**: Key areas of life affected

#### Personalized Insights

- **Life Area Impact**: Which areas of life are affected
- **Emotional Response**: Expected emotional reactions
- **Action Items**: Suggested responses and actions
- **Timing Advice**: When to act during the transit

### Transit Categories

#### Personal Planet Transits

- **Sun Transits**: Identity, self-expression, vitality
- **Moon Transits**: Emotions, instincts, daily rhythms
- **Mercury Transits**: Communication, learning, travel
- **Venus Transits**: Relationships, values, aesthetics
- **Mars Transits**: Energy, action, conflict

#### Social Planet Transits

- **Jupiter Transits**: Growth, opportunity, wisdom
- **Saturn Transits**: Structure, responsibility, karma
- **Uranus Transits**: Innovation, rebellion, awakening
- **Neptune Transits**: Spirituality, confusion, creativity
- **Pluto Transits**: Transformation, power, rebirth

### Transit Timing

#### Transit Phases

1. **Approach**: Transit building up (1-3 days before exact)
2. **Exact**: Peak influence (exact aspect)
3. **Separation**: Transit fading (1-3 days after exact)

#### Duration Guidelines

- **Fast Planets** (Sun, Moon, Mercury, Venus, Mars): 1-2 days
- **Social Planets** (Jupiter, Saturn): 1-2 weeks
- **Outer Planets** (Uranus, Neptune, Pluto): 1-2 years

## Troubleshooting

### Common Issues

#### Chart Not Loading

**Symptoms:** Blank chart display, error messages
**Solutions:**

1. Verify birth data accuracy
2. Check internet connection
3. Clear browser cache
4. Try different browser

#### Transit Delays

**Symptoms:** Transits not updating in real-time
**Solutions:**

1. Check system status at `/api/health`
2. Refresh the page
3. Clear application cache
4. Contact support if persistent

#### Agent Unresponsive

**Symptoms:** Agents not responding to messages
**Solutions:**

1. Check agent availability in System Monitor
2. Try different agent
3. Refresh conversation
4. Report via feedback system

#### Notification Issues

**Symptoms:** Not receiving expected notifications
**Solutions:**

1. Check notification preferences
2. Verify email/push permissions
3. Test notification channels
4. Review spam folders

## API Reference

### Authentication

All API requests require authentication:

```bash
# Using Bearer token
curl -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  https://api.planetary-agents.com/api/endpoint

# Using API key
curl -H "X-API-Key: YOUR_API_KEY" \
  https://api.planetary-agents.com/api/endpoint
```

### Core Endpoints

#### Natal Charts

```http
GET    /api/user-natal-charts          # List user charts
POST   /api/user-natal-charts          # Create new chart
GET    /api/user-natal-charts/:id      # Get specific chart
PUT    /api/user-natal-charts/:id      # Update chart
DELETE /api/user-natal-charts/:id      # Delete chart
```

#### Transit Analysis

```http
POST   /api/personalized-transits      # Calculate transits
GET    /api/natal-transits             # Get stored transits
POST   /api/transit-comparison         # Compare time periods
```

#### Agent Chat

```http
POST   /api/unified-multi-agent-chat   # Multi-agent conversation
GET    /api/agents                     # List available agents
GET    /api/agents/:id                 # Agent details
```

#### Notifications

```http
GET    /api/notifications              # Get notifications
POST   /api/transit-notifications      # Setup transit alerts
PUT    /api/notifications/:id/read     # Mark as read
```
